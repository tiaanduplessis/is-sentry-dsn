## Install

This project uses [node](https://nodejs.org) and [npm](https://www.npmjs.com). 

```sh
$ npm install is-sentry-dsn
$ # OR
$ yarn add is-sentry-dsn
```

## Usage

```js
import isSentryDSN from 'is-sentry-dsn'

isSentryDSN('foobarbaz') // false
```
